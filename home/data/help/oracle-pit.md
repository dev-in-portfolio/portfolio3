# Oracle Pit — Help

**A ‘decision pressure test’ app—built to showcase structured reasoning under uncertainty.**

## What this is
Oracle Pit is a portfolio app that frames decision‑making as a workflow: you provide constraints, it produces a structured set of options, risks, and next steps. The vibe is playful, but the core is serious: constraint capture, clarity, and traceable logic.

## How to use it
Enter the question or scenario, then add constraints (budget, timeline, risk tolerance, “must haves”). Run the analysis to get:
- a recommended path
- alternatives
- failure modes
- what information would change the answer

Treat it like a consulting worksheet. The point is structure, not mysticism.

## Tips
Good signals:
- It distinguishes facts vs assumptions.
- It gives you a “what would change my mind” list.
- It avoids false certainty.

If you want to stress test: use contradictory constraints and see if it flags the conflict.

## Troubleshooting
- **Output feels too long**: add a “brevity” or “executive summary” setting if available.
- **Feels too confident**: increase uncertainty signaling (confidence, unknowns) in future revisions.
- **Nothing loads**: Oracle Pit may be newly integrated; confirm the app link is wired in the Apps list.

## Privacy + data
If running locally/DEMO mode, data stays in the browser. If a live model is enabled later, disclose what is transmitted.
