# Prompt Vault — Help

**A prompt library + workflow organizer demo—built for speed, reuse, and clean categorization.**

## What this is
Prompt Vault is a portfolio app that treats prompts like assets: searchable, taggable, and reusable. It’s meant to show practical information architecture and “build for reuse” thinking.

## How to use it
Browse categories or search. Open a prompt, copy it, and adapt it. If the app supports saving or favorites, use it to keep your “go‑to” prompts in one place.

For recruiters: the interesting part is the UX and organization, not the specific prompt text.

## Tips
Look for:
- fast search and clear filtering
- predictable formatting when copying
- sensible defaults (tags, titles, categories)

If you’re building this out further: focus on export (Markdown/JSON) so prompts aren’t trapped in one UI.

## Troubleshooting
- **Copy button doesn’t work**: some browsers block clipboard without user interaction; try a long‑press select + copy.
- **Nothing shows**: confirm the seed data loaded; if you’re offline, a live fetch may fail—prefer baked data for portfolio demos.
- **Mobile scrolling**: if a modal is open, scroll inside the modal, not the background.

## Privacy + data
A prompt library can be sensitive. In demo mode, keep everything local. If you add accounts later, treat prompt text as user content with clear privacy posture.
