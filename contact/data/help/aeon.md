# AEON — Help

**What this is:** A cinematic, parameter-driven warp visualizer — the “quantum vibe” lab.

## Quick start
1. **Open the lab** and let it run for a few seconds (most visuals ramp in).
2. Use the **controls/panels** to change parameters.
3. Try **resize** (desktop ↔ narrow) and **interact** (mouse/touch) — most apps respond.

## Controls & interactions (common)
- **Mouse / touch:** hover / drag / tap to perturb or steer (lab-dependent).
- **Keyboard:** if you see a HUD hint, it’s real — otherwise assume mouse-only.
- **Reset:** refresh is always safe; these are built to be stateless on reload.

## Abilities demonstrated
- Procedural animation loop + perf-safe rendering
- Param controls (sliders/toggles) driving visuals
- Cinematic “feel” (lighting, glow, easing)
- Clean UI layering over realtime canvas

## If something looks “stuck”
- Give it **2–3 seconds** — several apps fade in intentionally.
- If the UI loads but visuals don’t: hard refresh and check browser console.
- Mobile Safari can be picky with high particle counts — try desktop if needed.
