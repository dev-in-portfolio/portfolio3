# STRING THEORY (4D Manifold Lab) - User Guide

STRING THEORY is an advanced 3D WebGL visualization tool designed to render and interact with 4D manifold simulations directly in your browser.

## Features
- **3D Viewport**: An interactive canvas where the 4D manifold is rendered. You can rotate, pan, and zoom the simulation.
- **Authored Presets**: Named parameter stories that make the lab easier to explore intentionally instead of only tweaking raw controls.
- **Compare Mode**: Capture two parameter states and re-apply them to compare topology, mode, density, and 4D mix.
- **Why This Matters**: A lightweight teaching layer that explains what the current manifold family and parameter balance are trying to show.
- **Post-Processing Effects**: Includes Screen Space Ambient Occlusion (SSAO) and Bloom for high-fidelity visuals.
- **Sidebar Controls**: Tweak rendering parameters, lighting, and simulation variables in real-time.
- **Chat/Log Interface**: Review system messages, simulation states, or interact with the lab assistant via the chat cards.

## Step-by-Step Usage
1. **Interact with the Canvas**: Click and drag on the central 3D viewport to rotate the manifold. Scroll to zoom in and out.
2. **Start from a Story**: Use the Authored Presets section to load a named manifold scenario before making local tweaks.
3. **Adjust Parameters**: Use the left sidebar to modify the simulation. You can adjust topology, mode, 4D mix, density, and post-processing effects.
4. **Compare Two States**: Capture Snapshot A and Snapshot B to compare how the visual story changes, then re-apply either state instantly.
5. **Read the Teaching Layer**: The “Why This Matters” panel explains the current family in plain language so the lab teaches as you interact.
6. **Monitor Status**: Keep an eye on the top right status pills and badges in the viewport for real-time performance metrics and simulation states.

## Troubleshooting
- **Sluggish Performance**: If the visualization is lagging, try turning off Bloom or SSAO in the sidebar controls to improve frame rates.
- **Mobile Issues**: On mobile devices, the canvas uses special touch-action settings. Use one finger to rotate and two fingers to zoom.
