# Transit - User Guide

Transit (TRANSIT 3D // PRESTIGE EDITION) is an immersive, interactive 3D exploration lab built around exoplanet transit discovery. It now behaves like a guided product, not just a scene viewer.

## Features
- High-quality 3D rendering using Three.js.
- Guided mission loop with step-by-step transit objectives.
- Criteria-aware checkpoint completion and a persistent mission log.
- Authored scene-state waypoints that jump the camera to meaningful perspectives.
- Context overlays explaining what the current geometry and light-curve state mean.
- Cinematic route replay built from the authored mission steps and waypoints.
- Interactive camera controls (pan, zoom, orbit).
- "Prestige Edition" dark-themed user interface.

## Step-by-Step Usage
1. Launch the Transit 3D application.
2. Wait for the 3D assets and environment to load completely.
3. Use the Mission Control panel to follow the active discovery route.
4. Clear checkpoints when the route conditions are met, or force them if you want to step through the authored sequence manually.
5. Use Cinematic Route to replay the current mission as a guided camera-and-system sequence.
6. Click waypoint buttons such as Observer Deck, Ingress Gate, or Stellar Face to jump between authored scene states.
7. Watch the Context Overlay and viewport story card to understand why the flux trace is changing.
8. Click and drag to orbit the camera around the 3D scene.
9. Scroll to zoom in and out, or use touch gestures on mobile devices.
10. Interact with the on-screen controls to change system, noise, speed, size, impact, and visual mode.

## Troubleshooting
- If the screen is black, ensure your browser and device support WebGL.
- Try refreshing the page if the 3D scene fails to load.
- Performance issues can often be mitigated by closing other resource-intensive applications or browser tabs.
