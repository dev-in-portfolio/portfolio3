# LingoLive AI - User Guide

LingoLive AI (Mastery Edition) is a language practice application focused on turning conversation into repeatable coaching. It supports real-time audio interaction, deterministic demo turns, transcript review, and mastery follow-up.

## Features
- Real-time AI language conversation practice.
- Demo mode for instant trial without an API key.
- Option to provide your own API key for unlocked capabilities.
- Transcript review with correction markup after practiced lines.
- Mastery Lab follow-up so corrected phrases can become repeat drills.
- Clean, modern interface tailored for voice interaction.

## Step-by-Step Usage
1. Open the LingoLive AI app.
2. On the introductory modal, choose "Try Demo" to experience the UI, or enter your API key to unlock full features.
3. Select your target language and scenario.
4. Use your microphone to speak, and wait for the AI to respond.
5. Review the **Transcript Review** panel after each coached line. It shows your original wording, the coach version, and what changed.
6. Add useful corrections to the **Mastery Lab** when you want to practice them again later.

## Troubleshooting
- If the app cannot hear you, ensure you have granted microphone permissions in your browser.
- If the AI does not respond, verify your API key is valid or check your internet connection.
- If transcript review looks empty, run at least one coached turn first. The review panel only appears once the app has feedback for a user line.
