# SleepyStory — README

**A bedtime story generator product demo with kid‑safe defaults and simple controls.**

## Overview
SleepyStory showcases how to build a creative generator as a product: simple inputs, controlled outputs, and repeatable iteration.

## What it demonstrates
- **Delightful UX** with minimal configuration.
- **Tone and age control** for safety.
- **Iteration loop**: regenerate without losing settings.
- **Optional visuals** (where supported) with graceful fallback.

## Project structure
Key folders:

- `apps/sleepystory/`
- `shared/`
- `data/`


## Run locally
Serve locally and open `/apps/sleepystory/`. If image generation requires a provider, keep DEMO visuals enabled for portfolio testing.

## Deploy
Static deploy works for demo builds. If you add a backend later, keep it optional; the app should still be enjoyable without accounts.

## Notes
For kid‑focused content, consistency and safety matter more than cleverness. Design defaults so a parent can click one button and get something acceptable.
