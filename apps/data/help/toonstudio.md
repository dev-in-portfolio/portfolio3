# ToonStudio — Help

**A playful ‘make a cartoon’ studio demo—built to show creative tooling UX and iteration loops.**

## What this is
ToonStudio is a creative portfolio app that focuses on the “studio” experience: choose a style, tweak a character/scene, iterate, and export. It’s meant to be fun while still demonstrating product instincts: controls that make sense, previews that update, and safe fallbacks.

## How to use it
Pick a project or start new. Choose a style (if offered) and then modify the key controls (character, background, mood, props). Generate or preview results, then iterate.

On mobile: start with fewer controls visible, generate a baseline, then refine. Creative tools can overwhelm quickly—this app should not.

## Tips
Try evaluating it like a tool:
- Are the controls named clearly?
- Do changes produce predictable results?
- Is it easy to undo/regenerate without losing everything?

If this includes Pixar‑like styling: treat it as an aesthetic demo; the portfolio value is workflow design.

## Troubleshooting
- **Nothing generates**: some builds run in DEMO mode without providers; check for sample outputs.
- **UI feels cramped**: rotate to landscape or collapse advanced panels.
- **Export blocked**: some browsers block downloads from iframes; open in a new tab.

## Privacy + data
In demo mode, assets are local. If you add cloud saves later, make it explicit and opt‑in.
