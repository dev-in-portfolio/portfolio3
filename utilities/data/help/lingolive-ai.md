# LingoLive AI — Help

**A language practice assistant demo—designed as a real workflow, not a toy translator.**

## What this is
LingoLive AI is built around practical language use: writing prompts, conversation drills, and corrections that teach. It’s a portfolio piece that emphasizes UX: clear feedback, progressive difficulty, and user control over tone and level.

## How to use it
Choose a target language and your level. Enter a phrase or prompt, then generate:
- a corrected version
- an explanation (simple → detailed)
- practice questions and a short drill

If the app includes voice or audio features, treat them as optional—core value should work with plain text.

## Tips
Try:
- Ask for the same message in three tones (formal, casual, slang) and check consistency.
- Make an intentional mistake and see if it explains the *rule*, not just the correction.
- Switch levels and see whether feedback complexity changes appropriately.

## Troubleshooting
- **It ‘overcorrects’**: select a different style (literal vs natural). Good language tools should let the user choose.
- **Mobile keyboard covers controls**: collapse the keyboard and use the action buttons first.
- **No live AI**: some builds run in DEMO mode for keyless testing.

## Privacy + data
Text stays in the browser in DEMO mode. If live providers are enabled in the future, the app should disclose what content is sent for processing.
