# Myths & Facts Almanac — Help

**Live app:** https://dev-in-portfolio-myths-facts-almanac.netlify.app

## Quick start
1. Open the app and confirm the primary screen loads without console errors.
2. Run one complete user flow from entry to completion.
3. Verify responsive behavior on mobile and desktop widths.

## Runtime checklist
- Confirm routing and deep-link behavior.
- Verify external links and share/open actions.
- Check loading, empty, and failure states.
- Validate keyboard focus and basic accessibility labels.

## If something breaks
- Hard-refresh to clear stale assets.
- Re-test with browser extensions disabled.
- Capture exact step, URL, and console error for triage.
